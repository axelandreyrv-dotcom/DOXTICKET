# Code of Conduct

## English

DoxTicket aims to be a respectful, practical, and welcoming open source project.

Expected behavior:

- Be respectful and constructive.
- Assume good intent, but be clear about risks.
- Keep discussions focused on the project.
- Do not harass, insult, threaten, or discriminate.
- Report security issues privately through `SECURITY.md`.

Maintainers may moderate issues, discussions, pull requests, or participation when behavior harms the project or its contributors.

## Espanol

DoxTicket busca ser un proyecto open source respetuoso, practico y acogedor.

Comportamiento esperado:

- Ser respetuoso y constructivo.
- Asumir buena intencion, pero ser claro con los riesgos.
- Mantener discusiones enfocadas en el proyecto.
- No acosar, insultar, amenazar ni discriminar.
- Reportar problemas de seguridad en privado segun `SECURITY.md`.

Los mantenedores pueden moderar issues, discusiones, pull requests o participacion cuando el comportamiento dane al proyecto o a sus contribuidores.
