# Contributing to DoxTicket

## English

DoxTicket is an open source self-hosted helpdesk. Early contributions should focus on bugs, security, documentation, tests, and stability.

Before contributing:

- Read `AGENTS.md`.
- Read the relevant files in `Docs/`.
- Do not add billing, cloud-hosted promises, plugins, or subtickets unless a documented decision exists.
- Preserve tenant isolation by `company_id`.
- Preserve `Powered by DoxTicket`.
- Add tests for critical behavior.

Security issues must be reported privately. See `SECURITY.md`.
